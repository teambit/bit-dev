# Code Editor Setup

Use any standard IDE and tools to edit code and text on the Bit workspace.

All base templates comes built in with an example `launch.json` for VSCode Debugging Configuration.

### Debugging in VSCode

[Base Templates](/dev-services/generator/generator-overview) comes built in with an example `launch.json` for [VSCode Debugging Configuration](https://code.visualstudio.com/docs/editor/debugging).

```json
{
  // Use IntelliSense to learn about possible attributes.
  // Hover to view descriptions of existing attributes.
  // For more information, visit: https://go.microsoft.com/fwlink/?linkid=830387
  "version": "0.2.0",
  "configurations": [
    {
      "type": "node",
      "request": "launch",
      "name": "test",
      "program": "${workspaceFolder}/node_modules/@teambit/bit/dist/app.js",
      "args": ["test"],
      "resolveSourceMapLocations": ["${workspaceFolder}/node_modules/@my-org/**/*.js"],
      "outFiles": ["${workspaceFolder}/node_modules/@my-org/**/*.js"],
      "console": "integratedTerminal",
      "sourceMaps": true,
      "internalConsoleOptions": "neverOpen",
      "cwd": "${workspaceFolder}"
    }
  ]
}
```
