A card component that is used to in the featured carousel of the quick-start component.

### Component usage

```js
<Card
  heading="Featured"
  description="This is a featured card."
  frameworkLogos={[
    {
      src: "https://static.bit.dev/extensions-icons/react.svg",
      inactiveSrc: "https://static.bit.dev/extensions-icons/react_grey.svg",
      alt: "React",
      href: "/",
    },
  ]}
/>
```
