This component is a starter blog component designed to be used as a
starting point for your blog. The most straightforward way to use it is to
add it to your workspace and start writing your blog! You can find
more information about the starter blog component in the [component's
documentation](https://bit.dev/teambit/blog/starter/starter-blog).

In this tutorial, we will learn how we composed this starter blog out
of several independent components. By the end of this tutorial, you will
have a fully functional blog website!
