Theme Provider helps product teams collaborate on design tokens and variables and get them to agree on a single, unified schema.


### Component Usage

```ts
const ThemeTokens = {
  backgroundColor: 'white'
};

const { ThemeProvider, useTheme } = createTheme(ThemeTokens);
```

### Create a basic theme

Modify the text to see it change live:

```js
const themeTokens = {
  backgroundColor: 'white'
};

const { ThemeProvider, useTheme } = createTheme(themeTokens);
```

### Create a light theme

Modify the text to see it change live:

```js
const lightTokens = {
  backgroundColor: 'black'
};

const { ThemeProvider, useTheme } = createTheme(ThemeTokens);
```

### Create a dark theme

Modify the text to see it change live:

```js
const darkTokens = {
  backgroundColor: 'white'
};

const { ThemeProvider, useTheme } = createTheme(ThemeTokens);
```
