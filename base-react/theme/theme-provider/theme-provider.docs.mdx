Theme Provider generates a theme out of design tokens (for example: `{ colorText: '#cab854' }`).
The generated theme component styles all descendant components of it.

## Basic usage

Use `createTheme` to generate a new theme out of a list of design tokens:

```ts
import { createTheme } from '@teambit/base-react.theme.theme-provider';

/* describe the theme's design tokens and set their default values */
export const defaultDesignTokenValues = {
  colorBackground: '#ffffff',
  colorText: '#cab854',
};

/* generate a theme with the above default values */
export const Theme = createTheme({
  theme: defaultTokenValues,
});

/* create a theme schema to standardize future customized themes */
export type ThemeSchema = typeof defaultDesignTokenValues;
```

The generated theme can be used to theme [components styled with CSS](https://bit.dev/teambit/base-react/theme/theme-provider/~code/examples/button-styled-with-css.demo.tsx) as well as [components styled with CSS-in-JS](https://bit.dev/teambit/base-react/theme/theme-provider/~code/examples/button-styled-with-js.demo.tsx).

## Composing a new theme

Create an object with new design token values:

```ts
/* the theme schema is used to maintain consistency in the way design tokens are used */
import type { ThemeSchema } from './examples/default-theme.demo';

export const darkThemeOverrides: ThemeSchema = {
  colorBackground: '#474747',
  colorBorder: '#30302e',
};
```
