import { Button } from './button';

A button component that can be used as a `<button>` or `<a>` element. Includes a prop for secondary button styling.

### Primary Button

```js live
<Button>Get Started</Button>
```

### Secondary Button

```js live
<Button secondary>Watch video</Button>
```

### Link Button

```js live
<Button href="http://twitter.com">Getting Started</Button>
```
