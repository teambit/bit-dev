import { Tooltip } from '@teambit/community.ui.text-tooltip';

import { Heading } from './heading';
import { Elements } from './elements';

Renders text as a heading element (`H1` to `H6`) with Bit's heading style.

### Basic usage

```jsx live
<Heading>Build anything in components</Heading>
```

### Choosing the heading element type

Pass the `element` prop to choose the element type to return (`H1` to `H6` of the <Tooltip content={importElements} language='typescript'>`Elements` enum</Tooltip>).

```jsx live
<Heading element={Elements.H3}>Build anything in components</Heading>
```

### Overriding heading default size

Pass the `size` prop to override the heading element default size.

```jsx live
/* an H2 element rendered with H4 size */
<Heading element={Elements.H2} size="h4">
  Build anything in components
</Heading>
```
