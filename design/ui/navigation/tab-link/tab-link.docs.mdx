import { TabLink } from './tab-link';

Tab link is used to display a navigation link that shows as active if `href` matches `pathname`, usually used in headers or tabs to display a specific active ui.

It uses `@teambit/base-react.navigation.link` under the hood for its functionallity.

```js live
<TabLink>hello</TabLink>
```
