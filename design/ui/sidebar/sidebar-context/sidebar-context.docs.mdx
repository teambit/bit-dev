## React Theme Context

This is a simple [React Context](https://reactjs.org/docs/context.html) shared as a Bit component.

Use this component to access the sidebar's status and control as a context to be used in it's children.

### Component usage

Wrap your app in the `SidebarProvider`

```tsx
() => {
  import React from 'react';
  import { SidebarProvider } from '@teambit/design.ui.sidebar.sidebar-context';

  // My lovely children now get the context!
  <SidebarProvider>
    <SidebarComponent />
  </SidebarProvider>;
};
```

### Using the context 

The children can now access the context.

```tsx
import React, { useContext } from 'react';
import { SidebarContext } from '@teambit/design.ui.sidebar.sidebar-context';

function SidebarComponent() {
  // sidebar component can access the context.
  const { isOpen, setIsOpen } = useContext(SidebarContext);
  return <div></div>;
}
```
