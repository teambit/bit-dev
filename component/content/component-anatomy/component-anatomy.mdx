# Component anatomy

A Component contains 5 types of data:

- Source code
- Dependencies
- Configuration
- Artifacts
- Metadata

## Source code

The Component's main file, implementation files and development files. A file is considered part of the Component's files if it is under the same Component Directory.

```bash
$ bit show teambit.community/ui/graph/bubble-graph
```

## Artifacts

Artifacts are generated during the Component build. Artifacts are persisted when a Component is snapped or tagged. Each Component version holds the artifacts that were generated as part of its build.

The following lists a Component's latest artifacts:

```bash
bit artifacts teambit.community/ui/graph/bubble-graph
```
