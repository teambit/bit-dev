The bit-dev app is a deployable component that provides Bit's community with documentation, guides and other Bit resources.
The app is composed of open-source components from various scopes.

### Run the application

Import the app and add the following config to your `workspace.jsonc`:

```json
"teambit.community/apps/bit-dev": {},
```

To run the app locally, run the following:

```bash
bit run bit-dev
```
