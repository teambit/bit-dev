Complete section component for the Features to be used in a page component. This component contains:

- the section title
- a grid of features cards

### Section showing Features with imported Json

Imports mock data in json format from the entity component which contains an array of features.

```js
import { features } from '@teambit/community.entity.features';

<FeaturesSection features={features} title="Build better. Build scalable. " />;
```
