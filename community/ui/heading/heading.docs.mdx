Renders text as a heading with Bit Community styles.

### Basic usage

```jsx live
<Heading>Build anything in components</Heading>
```

### Highlighting a heading

```jsx live
<Heading element="h1" highlight>
  Build anything in components
</Heading>
```

### Highlight any text using the highlight class

```js live=true
<span className={highlightClass}>highlighted text</span>
```

### Choosing the heading element type

Pass the `element` prop to choose the element type to return (`H1` to `H6` of the <Tooltip content={importElements} language='typescript'>`Elements` enum</Tooltip>).

```jsx live
<Heading element={Elements.H3}>Build anything in components</Heading>
```

### Overriding the heading size

You can override the size of the `Heading` using the existing pallete. In this case, the output is a `H2` element with the size of an `H6` element according to the _Design scope_ pallete.

```js live=true
<Heading element="h2" size="h6">
  Build anything in components
</Heading>
```
