Component for the main Heading.

### Heading with H1 element

```js live=true
<Heading element="h1">Build anything in components</Heading>
```


### Highlighted heading with H1 element
```js live=true
<Heading element="h1" highlight>Build anything in components</Heading>
```

### Override heading size

You can override the size of the `Heading` using the existing pallete. In this case, the output is a `H2` element with the size of an `H6` element according to the *Design scope* pallete.
```js live=true
<Heading element="h2" size="h6">Build anything in components</Heading>
```
