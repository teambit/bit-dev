import { Terminal } from './terminal';

A terminal that renders some code examples.

### Component Usage

```js
<Terminal content="hello from Terminal" />
```

### Using props to customize the text and animate it.

Modify the text to see it change live:

```js live
<Terminal content="hello from Terminal" animate={false} delay={140} />
```
