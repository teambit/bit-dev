Community Highlighter helps you discover components by revealing the component IDs of UI elements in a page.
The highlighter data is injected into the component's bundle <!-- TODO: change the link to the relevant page in the docs --> using [Bit's babel plugin](https://bit.dev/teambit/react/babel/bit-react-transformer).
This plugin is used by default in applications built with the [React app-type](https://bit.dev/teambit/react/react).

## Basic usage

Install the Community Highlighter and wrap your components with it to enable inspection.

<LeftRight
  left={<ExampleSnippet />}
  right={<Example />}
  style={{ marginTop: '1rem' }}
/>

## Disabling the highlighter using the toggler component

Place the Toggler anywhere, nested inside the Community highlighter. The highlighter's state (`disable: boolean`) is persisted in the local storage.

<LeftRight
  left={<TogglerSnippet />}
  right={<ExampleWithToggler />}
  style={{ marginTop: '1rem' }}
/>

The highlighter's state (disable: boolean) is persisted in the local storage.

### Disabling the highlighter using the useHighlighter hook

Use the `useHighlighter` in a component nested inside the Community highlighter. The highlighter's state (`disable: boolean`) is persisted in the local storage.

<LeftRight
  left={<UseHighlighterSnippet />}
  right={<ExampleWithUseHighlighterHook />}
  style={{ marginTop: '1rem' }}
/>
