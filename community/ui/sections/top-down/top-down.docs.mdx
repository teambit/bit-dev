'Top Bottom' is a page section that renders a title, a subtitle, and other elements, from top to bottom.

## Basic usage

```jsx live
  <TopDown
    title="Create. Collaborate"
    subtitle="Create new components quickly and compose them with others."
  >
    <button>a custom button</button>
  </TopDown>
```
