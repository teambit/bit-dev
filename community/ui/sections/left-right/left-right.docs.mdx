'Left Right' is a page section that renders a title, a paragraph, and other elements, from left to right.

## Basic usage

```jsx live
  <LeftRight
    title="Create. Collaborate"
    text="Create new components quickly and compose them with others."
  >
    <button>a custom button</button>
  </LeftRight>
```
