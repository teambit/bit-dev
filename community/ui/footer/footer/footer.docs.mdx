import { Footer } from './footer';
import { footerMock } from './footer.mock';

A concrete footer for the bit.dev app. Accepts a list of categories with a list of links for each category.

Each link can be displayed with or without an icon.

```ts live
<Footer
  categoryList={[
    {
      title: 'links with icons',
      links: [
        {
          icon: 'https://static.bit.dev/bit-logo.svg',
          text: 'Bit',
          href: 'https://bit.dev',
          external: true,
        },
        {
          icon: 'slack',
          text: 'Slack',
          href: 'https://bit.dev',
          external: true,
        },
        { icon: 'twitter-logo', text: 'Twitter', href: '#' },
        { icon: 'github-logo', text: 'Github', href: '#' },
      ],
    },
    {
      title: 'links without icons',
      links: [
        {
          text: 'Bit',
          href: 'https://bit.dev',
          external: true,
        },
        {
          text: 'Slack',
          href: 'https://bit.dev',
          external: true,
        },
        {
          text: 'Twitter',
          href: '#',
        },
        { text: 'Github', href: '#' },
      ],
    },
  ]}
/>
```

The footer also provides a fallback bottom section if no children are provided.

To override this behavior, provide `children` to override it.

```ts live
<Footer categoryList={footerMock}>
  <div>override bottom line</div>
</Footer>
```
