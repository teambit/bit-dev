The bubble-highlighter highlights components using a bubble-like border. Optionally, it can include labels that indicate the component name and scope.

### Basic highlighting

Wrap the component to be highlighted with the bubble highlighter.  
Pass a [`componentId`](https://bit.dev/teambit/component/component-id/~code/component-id.ts) object to provide the bubble highlighter with information regarding the highlighted component.

<LeftRight
  left={<HighlightButtonSnippet />}
  right={<BasicBubbleHighlighterExample />}
/>

### Highlight with labels

Pass the `showId` prop (boolean) to display the scope and name of the component being highlighted.

(Try clicking on the labels to load the component/scope page.)
