import { BubbleHighlighter } from './bubble-highlighter';
import { ComponentID } from '@teambit/component-id';
import { Button } from '@teambit/design.ui.buttons.button';

Bubble highlights any component rendered inside.

### Bubble Highlighting a Button component

```js live
<BubbleHighlighter
  componentId={ComponentID.fromString('teambit.design/ui/buttons/button')}
  style={{ width: 170 }}
>
  <Button>Getting Started</Button>
</BubbleHighlighter>
```
