A "sticky" menu that stays on the screen as you scroll down ("sticks" to the top when its offset position meets the viewport).

## Basic usage

Pass the `links` prop with an array of links. Use the `className` prop to customize the menu.

```js live
<StickyMenu
  className="my-class"
  links={[
    {
      href: '#compose',
      text: 'Compose'
    },
    {
      href: '#collaborate',
      text: 'Collaborate'
    },
    {
      href: '#release',
      text: 'Release'
    },
  ]}
/>
```
