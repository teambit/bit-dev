This component displays ['use-cases-card'](https://bit.dev/teambit/community/ui/use-cases/use-cases-card) components in a grid layout of two columns.


## Basic usage

Pass an array of  [UseCase](https://bit.dev/teambit/community/entity/use-cases) objects (containing use-cases info) to the `useCases` prop:

```js
import { UseCasesGrid } from '@teambit/community.ui.use-cases.use-cases-card-grid';
import { useCases } from './use-cases';

<UseCasesCardGrid useCases={useCases} />;
```

Try this component for yourself by changing the `useCases` data:

```js live
<UseCasesCardGrid
  useCases={[
    {
      title: 'Component Library',
```
