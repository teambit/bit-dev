This component describes and demonstrates the use-case object. It serves as the source of truth for components sending and receiving that data.

### Create a new use-case object
Use this component to create a new use-case and validate it follows the right schema:

```ts
import { UseCases } from '@teambit/community.entity.use-cases';
import { title, text, image, alt, href } from './my-use-case-info';

const myUseCase = new UseCases(title, text, image, alt, href);
```

### Type data transferred via network requests

```ts
import type { UseCasesFromApi } from '@teambit/community.entity.use-cases';

async function fetchUseCase(): Promise<UseCasesFromApi> {
  // ...
}
```

### Test components with mock data

Use the mock data provided by this component to test your components (without performing network requests). See an example in [this composition](https://bit.dev/teambit/community/ui/use-cases/use-cases-card-grid/~code/use-cases-card-grid.composition.tsx).
