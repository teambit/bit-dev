This component describes and demonstrates the "use-case" object. It serves as the source of truth for components sending and receiving "use-cases" data.

### Test components with mock data

Use the mock data provided by this component to test your components:

<UsageExample code={usingMock} output={<RenderUseCasesMock />} />

### Create a new use-case object

Use this component to create a new use-case and validate it follows the right schema:

```ts
import { UseCases } from '@teambit/community.entity.use-cases';
import { title, text, image, alt, href } from './my-use-case-info';

const myUseCase = new UseCases(title, text, image, alt, href);
```
