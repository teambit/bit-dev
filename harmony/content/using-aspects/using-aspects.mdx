# Using Aspects

Aspects can be used to extend Bit with simple configurations, to both the Scope and the Workspace. Aspects can also be composed into Harmony applications.

You can find more [available aspects here](/plugins) and on the [bit.cloud](https://bit.cloud/components?env=aspect) search!

## Configuring Aspects

Configuring Aspects can be done across Bit configuration, from Workspaces and Scopes and on Components.
```json
{
  "myorg.dev/my-aspect": {}
}
```

## Listing aspects

List all components and the aspects they have configured.

```
bit aspect list
```

## Inspecting aspects

Inspect a specific component` aspects, inclduing metadata and config.

```
bit aspect get docs/apps-overview
```


## Updating aspects

Update components to a newer aspect version.

Update all components using the community-react aspect.

```bash
bit aspect update teambit.community/envs/community-react
```

Update components using the community-env aspect under the ui/* namespace.

```bash
bit aspect update teambit.community/envs/community-react "ui/*"
```

## Composing aspects into applications

```tsx
import { Harmony } from '@teambit/harmony';
import { MyAspect } from '@myorg/dev.my-aspect';

Harmony.run([MyAspect]);
```
