# Creating Aspects

Aspects are components used to extend Bit. Every Aspect is a component, which is using the Aspect environment.

<ComponentCardDisplay componentIds={['teambit.harmony/aspect']} style={{ float: 'left', margin: 20 }} />

## Creating an Aspect

Create a basic Aspect with using the official aspect env template.

```
bit create aspect my-aspect
```

Your new component incldues few files: `my-aspect.aspect.ts`, `my-aspect.main-runtime.ts` and regular documantation, compositions and test files.
