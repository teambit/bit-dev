A components that renders a group of components with a title.

### Component usage

```js live
<ComponentCardGroup
  title="My Components"
  components={[
    {
      id: 'teambit.harmony/aspect',
      description: 'Lorem ipsum dolor sit amet, consectetur adipiscing elit. Ipsum fdfd',
      envIcon: 'https://static.bit.dev/extensions-icons/default.svg',
      preview: <div>hello world</div>,
    },
  ]}
/>
```
