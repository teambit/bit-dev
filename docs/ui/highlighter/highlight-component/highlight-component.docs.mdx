A component to highlight React components in a component tree. Using this component you can highlight the entire tree, or just parts of it.

### Component usage
```js
<HighlightComponent>
  <ProductCard />
</HighlightComponent>
```

### Highlight all nested components in a component tree

```js live
<HighlightComponent title="Product card breakdown">
  <div>Product card</div>
</HighlightComponent>
```
