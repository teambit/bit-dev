## React Component for rendering doc page

A component that renders doc's page using MDX.

### Component usage

```js
<DocPage
  index={0}
  route={{ title: 'Create a new Workspace', component: <CreateWorkspace />, absPath: '/', plugins: [] }}
>
  <CreateWorkspace />
</DocPage>
```
