### Arguments

Mandatory arguments:

```
bit create COMPONENT_TEMPLATE COMPONENT_NAME
```

Arguments that can repeat:

```
bit create COMPONENT_TEMPLATE COMPONENT_NAME...
```

Optional arguments:

```
bit create COMPONENT_TEMPLATE COMPONENT_NAME [--scope SCOPE_NAME]
```

Mutually exclusive arguments:

```bash
bit tag {--all | COMPONENT_ID...}
```
